le programme min_chat.php est le résultat d'un exercice d'apprentissage du langage PHp suivi sur openclassroom!!!

on y trouve un formulaire de saisies avec pseudo et message.
les messages sont envoyé dans une base mysql

j'ai rajouté le nombre de message total ainsi que la possibilité de voir les anciens messages
